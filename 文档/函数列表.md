### void throwError(char *s, int row, int col)

#### 接受参数

1. char *s

    错误具体信息

2. int row

    出错行数

3. int col

    出错列数

#### 返回参数

无

#### 函数说明

抛出错误信息，并显示错误具体原因，错误发生行数与列数
